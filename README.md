# تطبيق رصد (نسخة محلية بدون إنترنت)

هذا مشروع جاهز (HTML/CSS/JS) يعمل كـ PWA ويحفظ البيانات محليًا داخل الجهاز عبر IndexedDB.

## 1) تشغيله على الكمبيوتر
1. فك الضغط.
2. افتح الملف `index.html` مباشرة بالمتصفح (Chrome / Edge).
3. أول تسجيل دخول:
   - المستخدم: `admin`
   - كلمة المرور: `1234`
4. غيّر كلمة المرور من: (إدارة الموظفين) أو أنشئ حسابًا جديدًا واحذف الافتراضي إذا رغبت.

> ملاحظة: بعض المتصفحات تمنع Service Worker عند فتح الملف مباشرة.
> الأفضل تشغيله عبر سيرفر محلي بسيط.

### تشغيله كسيرفر محلي (اختياري)
- على Windows: استخدم أي سيرفر بسيط أو VSCode Live Server.
- على macOS/Linux:
  ```bash
  python3 -m http.server 8080
  ```
  ثم افتح: `http://localhost:8080/rsd_offline_app/`

## 2) تثبيته على الجوال (PWA)
- افتح الرابط/السيرفر على الجوال.
- من المتصفح اختر: **Add to Home Screen / إضافة إلى الشاشة الرئيسية**.

## 3) استيراد الطلاب (الملف المرفق من الإرشاد)
- من الصفحة: **الطلاب** → **استيراد ملف الإرشاد (المرفق)**
  - سيتم استيراد الملف الموجود في: `assets/StudentGuidance_clean.csv`

## 4) استيراد ملف جديد
- يدعم CSV مباشرة.
- لدعم Excel (xls/xlsx):
  1) ضع مكتبة SheetJS محليًا داخل `vendor/xlsx.full.min.js`
  2) افتح `index.html` وأزل التعليق عن السطر:
     ```html
     <script src="vendor/xlsx.full.min.js"></script>
     ```
  3) بعد ذلك سيقبل الاستيراد من Excel أيضًا.

## 5) النسخ الاحتياطي
- الإعدادات → تصدير نسخة JSON
- لاسترجاعها: الإعدادات → استيراد نسخة JSON

## 6) ربط SMS (المدار التقني / MobileNet)
- الإعدادات → تفعيل SMS
- أدخل بيانات الاشتراك (userName / userPassword / userSender / apiKey إن وجد)
- ملاحظة: قد يمنع مزود الرسائل CORS؛ عندها استخدم وضع Ping أو وفّر Backend وسيط.
