# yaqubi-rsd (Offline PWA)

هذا مشروع صفحة ويب تعمل بدون إنترنت (PWA) لمتابعة الطلاب (مواظبة/سلوك) وتخزين البيانات داخل الجهاز عبر IndexedDB.

## التشغيل على macOS
> مهم: لا تفتح index.html بالنقر المزدوج (لن يعمل الـ Service Worker بشكل صحيح). شغّلها عبر سيرفر محلي.

### خيار سريع (Python)
1) افتح Terminal داخل مجلد المشروع
2) نفّذ:
   python3 -m http.server 8080
3) افتح المتصفح على:
   http://localhost:8080/

### خيار VS Code
- افتح المجلد في VS Code
- ثبّت إضافة Live Server
- اضغط Go Live

## الاختبار بدون إنترنت
- افتح الصفحة مرة واحدة وهي متصلة
- ثم افصل الإنترنت، وستعمل لأن الملفات مُخبأة بواسطة sw.js

## استيراد الطلاب
- من الإعدادات > استيراد الطلاب
- يدعم CSV. يوجد ملف نموذجي داخل assets/StudentGuidance_clean.csv
