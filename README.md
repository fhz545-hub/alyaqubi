# تطبيق ثانوية اليعقوبي – متابعة المواظبة والسلوك (Offline PWA)

## المزايا
- يعمل دون إنترنت (بعد فتحه أول مرة) + قابل للتثبيت كتطبيق على الجوال/الكمبيوتر (PWA).
- حفظ محلي دائم (IndexedDB) لكل طالب: مواظبة + سلوك + سجل العمليات.
- البحث لا يُظهر أسماء الطلاب إلا بعد إدخال بحث (خصوصية).
- رسائل: حفظ بالقائمة ثم إرسال لاحقًا (SMS/واتساب).
  - واتساب: يفتح الرسائل واحدة تلو الأخرى.
  - SMS: يعتمد على رابط مزود الرسائل (URL Template) ويحتاج إنترنت.

## التشغيل على GitHub Pages (موصى به)
1) ارفع محتويات هذا المجلد كما هي في مستودع GitHub.
2) من Settings → Pages → اختر Branch: main و Folder: /(root).
3) افتح الرابط الناتج، ثم من المتصفح اختر (Add to Home Screen) للتثبيت على الجوال.

## استيراد الطلاب
- من الإعدادات → استيراد الطلاب:
  - يدعم CSV (الأفضل).
  - يوجد ملف نموذجي مرفق: assets/StudentGuidance_clean.csv

> ملاحظة: Excel (XLS) غير مدعوم داخل المتصفح دون مكتبة إضافية. إن كان ملفك XLS، حوّله إلى CSV ثم استورده.

## مزود الرسائل (SMS)
من الإعدادات ضع:
- قالب رابط الإرسال (URL Template) بصيغة:
  `https://api.example.com/send?u={username}&p={password}&sender={sender}&to={numbers}&msg={message}`

ثم أدخل:
- Sender / Username / Password
- اختر إرسال جماعي أو فردي حسب ما يدعمه المزود.

> أمان مهم: لا تضع بيانات المزود في مستودع عام؛ الإعدادات هنا تُحفظ محليًا على الجهاز فقط.
