# راصد - نسخة دون إنترنت (ثانوية اليعقوبي)

هذه الحزمة تعمل كـ PWA (تفتح على الجوال والكمبيوتر) وتُخزّن البيانات داخل الجهاز (IndexedDB).

## أهم نقطتين
- **بيانات الطلاب لا تظهر في الصفحة** إلا عند البحث (خصوصية).
- **قاعدة الطلاب ** مرفقة داخل `assets/students_seed.json` ويتم تحميلها تلقائياً عند أول تشغيل إذا كانت قاعدة الطلاب فارغة.

## الاستيراد (CSV)
الأعمدة المطلوبة:
- اسم الطالب: `اسم الطالب` أو `الاسم` أو `name`
- رقم الهوية: `رقم الهوية` أو `الهوية` أو `idNumber` أو `NationalID`
- الصف + الشعبة (مستحسن للكشوف)

## الطباعة
- كشف أسبوعي (الأحد → الخميس) مع **عمود باركود الهوية**.
- كروت صغيرة (85×55mm) للتأخر/الغياب/السلوك.

## الرسائل SMS (اختياري)
لأمان التوكن استخدم Cloudflare Worker كـ Proxy (انظر `cloudflare_worker_sms_proxy.js`).


## استيراد بيانات الطلاب
- افتح التطبيق ثم اضغط (استيراد الطلاب) واختر ملف CSV/XLSX.
- قالب الأعمدة موجود في: assets/students_template.csv
